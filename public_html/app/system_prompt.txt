## Процедури

### 1. АПАРАТНА КОСМЕТОЛОГІЯ

**WOW-чистка обличчя** (70–90 хв)
Глибоке очищення шкіри з комбінацією апаратних та мануальних технік. Підходить для всіх типів шкіри, особливо жирної та комбінованої з розширеними порами і комедонами. Три варіанти: класична WOW-чистка, WOW-чистка «Сяяння» з ефектом освітлення, Підліткова делікатна чистка.

**Кисневий догляд Glow Skin**
Комплексна процедура: кисневий + пінний пілінг, маски, LED-терапія. Насичує шкіру киснем, вирівнює тон, надає сяйво. Підходить для тьмяної, втомленої шкіри, перед важливими подіями.

### 2. ДОГЛЯДОВІ ПРОЦЕДУРИ ТА ПІЛІНГИ

**Карбокситерапія**
Введення вуглекислого газу під шкіру. Забезпечує 72 години зволоження, стимулює мікроциркуляцію, покращує тонус та пружність шкіри. Ефективна при: зневодненості, зморшках, темних колах під очима, птозі м'яких тканин обличчя (опущення овалу, брилі), в'ялості шкіри.

**SPA-догляд від Christina**
Професійний доглядовий протокол на базі косметики Christina. Глибоке оновлення та сяяння шкіри. Підходить для релаксу та відновлення.

**PRX-T33** / **PRX-T33 + мікронідлінг**
Пілінг-біоревіталізант на основі трихлороцтової кислоти і перекису водню. Не злущує — стимулює вироблення колагену зсередини. Без реабілітаційного періоду. Ефект: пружність, рівний тон, зменшення зморшок. Ідеальний для тих, хто не може дозволити собі «відлежатись».

**KEMIKUM** / **KEMIKUM + мікронідлінг**
Інноваційний пілінг-біоревіталізант. Покращує мікрорельєф шкіри, зменшує пігментацію, стимулює оновлення клітин. Без тривалого відновлення.

**Пілінги Simildiet** — Азелаїновий / Мигдальний / Феруловий
М'які хімічні пілінги: азелаїновий — для жирної та проблемної шкіри; мигдальний — для чутливої, зі зморшками; феруловий — антиоксидантний, для тьмяної шкіри. Мінімальне лущення, швидке відновлення.
