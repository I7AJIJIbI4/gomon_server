Політика конфіденційності

Дата оновлення: 01.04.2026

Ця Політика конфіденційності описує порядок збору, використання та захисту персональних даних клієнтів косметологічної клініки Dr. Gomon Cosmetology (далі — «Клініка»), які використовують веб-сайт gomonclinic.com, мобільний додаток (PWA/Android) та пов'язані сервіси.

1. Які дані ми збираємо

  • Контактні дані: ім'я, прізвище, номер телефону — для ідентифікації та зв'язку
  • Дані авторизації: OTP-коди (одноразові паролі) — для безпечного входу в додаток
  • Історія записів: дати, час, назви процедур, спеціаліст — для відображення в особистому кабінеті
  • Push-підписки: токен підписки Web Push — для надсилання нагадувань про записи та акції
  • Telegram ID: ідентифікатор Telegram-акаунту — для надсилання сповіщень через Telegram-бот
  • Технічні дані: IP-адреса, тип браузера, ОС — автоматично при відвідуванні сайту
  • Дані з Instagram: повідомлення з Direct, публічний профіль — через офіційний API Meta
  • Локальне сховище: токен сесії, кешовані записи — зберігаються на пристрої користувача (localStorage)

2. Як ми використовуємо дані

  • Авторизація в додатку та на сайті
  • Відображення записів, прайс-листу та особистого кабінету
  • Надсилання нагадувань про записи через Push, Telegram або SMS
  • Надсилання підтверджень створення та скасування записів
  • Інформування про акції та спеціальні пропозиції (за згодою)
  • Відповіді на запити через Instagram Direct
  • Покращення якості обслуговування та роботи додатку
  • AI-помічник для підбору процедур (дані не зберігаються на зовнішніх серверах AI)

3. Канали доставки сповіщень

Клініка використовує три канали для сповіщень клієнтам:

  • Web Push: через підписку в додатку (можна відключити в налаштуваннях браузера)
  • Telegram: через бот @DrGomonConciergeBot (підключається добровільно)
  • SMS: як резервний канал, якщо Telegram недоступний

Ви можете відмовитись від будь-якого каналу: відключити Push в налаштуваннях браузера, заблокувати Telegram-бот, або повідомити нас про відмову від SMS.

4. Інтеграція з Meta (Instagram)

Клініка використовує офіційний API Instagram для читання та відповіді на Direct-повідомлення. Ми маємо доступ до:

  • Тексту повідомлень
  • Публічної інформації профілю
  • Ідентифікаторів акаунтів

Автоматичних відповідей не використовується. Всі повідомлення обробляються спеціалістами клініки.

5. Мобільний додаток (PWA / Android)

Додаток Dr. Gomon доступний як Progressive Web App (PWA) та в Google Play. Додаток:

  • Зберігає токен сесії в localStorage пристрою для автоматичного входу
  • Кешує дані записів та прайс-листу для офлайн-доступу
  • Запитує дозвіл на Push-сповіщення (за згодою користувача)
  • Не збирає дані геолокації, камери чи мікрофону без явного дозволу
  • Голосовий ввід (Web Speech API) обробляється браузером локально, аудіо не передається на сервер клініки

6. Захист даних

  • Персональні дані зберігаються на захищеному сервері в Україні
  • Доступ до бази даних мають тільки авторизовані адміністратори клініки
  • Зв'язок між додатком та сервером захищений протоколом HTTPS
  • OTP-коди дійсні протягом 5 хвилин і автоматично видаляються
  • Сесії автоматично закінчуються через 30 днів неактивності
  • Журнал сповіщень автоматично очищується кожні 30 днів

7. Передача даних третім сторонам

Ми не продаємо і не передаємо персональні дані третім сторонам, крім:

  • SMS-провайдер (sms-fly.ua): номер телефону — для доставки SMS-повідомлень
  • Telegram Bot API: Telegram ID — для доставки сповіщень
  • Web Push сервіси (FCM/APNs): токен підписки — для доставки Push-сповіщень
  • WLaunch CRM: дані записів — для синхронізації з системою управління клінікою

Всі треті сторони використовують дані виключно для доставки повідомлень, не зберігають їх для власних цілей.
